# 转换规则:

## 1. 组件映射:

| C# WinForms 控件 | Shadcn-vue 组件             | 备注                            |
| ---------------- | --------------------------- | ------------------------------- |
| Button           | `<Button>`                  | 保留 Text 作为按钮文字          |
| TextBox          | `<Input>`                   |                                 |
| Label            | `<Label>` 或 `<span>`       | 纯展示用 span，表单关联用 Label |
| ComboBox         | `<Select>` 系列             | 包含 Trigger/Content/Item       |
| DataGridView     | `<Table>` 系列              | 提取列定义生成表头              |
| GroupBox         | `<Card>` 或带标题的 `<div>` | Text 属性作为标题               |
| CheckBox         | `<Checkbox>`                |                                 |
| Panel            | `<div>`                     |                                 |
| 其他未列出       | `<div>` + 注释              | 注明原控件类型                  |

### 忽略的控件类型（不生成对应元素）:

- BindingNavigator 及其子项 (ToolStripButton, ToolStripSeparator, ToolStripTextBox, ToolStripLabel)
- BindingSource

### DataGridView 特殊处理:

- 提取 DataGridViewTextBoxColumn 的 HeaderText 生成表头
- 列宽转换为相对宽度类 (w-1/4, w-1/3 等)
- 生成空数据的表格骨架

## 2. 布局规则:

- **禁止** absolute/fixed 定位
- **必须** 使用 Flex 或 Grid 布局

### 布局推断逻辑:

1. Y 坐标差 < 15px → 同一行 (flex-row)
2. X 坐标差 < 15px → 同一列 (flex-col)
3. 网格排列 → Grid 布局
4. GroupBox/Panel 内部独立处理布局

### 间距与尺寸:

- 使用 gap-2, gap-4 等 Tailwind 间距类
- 宽度使用 w-full, w-1/2, w-auto 等响应式类
- 避免硬编码像素值

## 3. 代码规范:

- Vue 3 Composition API + `<script setup>`
- 导入路径: `@/components/ui/xxx`
- 原控件 Name → ref 名称或元素 id
- 表单控件预留 v-model 绑定
- 中文文本直接使用
