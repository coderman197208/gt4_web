# 为项目增加redis数据库订阅和访问功能：

## 背景介绍

有一个C++项目，它会把实时数据(tag)写入redis数据库，然后通过主题'RealDataChanged'发布消息，消息值是tag的名字
以下是C++程序示例：

```C++
// tag_name:tag的名字
// tag_value:tag的值(JSON STRING)
// g_redis:redis连接

// 将tag的值写入Redis数据库
// 注意：空状态也必须写成合法 JSON 字符串，不能写空串
// 例如：PlanInfo -> "null"，ALIGN_POS_TUBE_INFO -> "[]"
g_redis->set(tag_name, tag_value);

// 发布详细消息到'RealDataChanged'主题
g_redis->publish("RealDataChanged", tag_name);
```

如果上游需要直接按这个规则写 Redis，可以使用下面这段示例代码：

```C++
#include <string>

std::string NormalizeRealtimeTagValue(const std::string& tagName, const std::string& rawValue)
{
	if (!rawValue.empty())
	{
		return rawValue;
	}

	if (tagName == "ALIGN_POS_TUBE_INFO")
	{
		return "[]";
	}

	if (tagName == "PlanInfo")
	{
		return "null";
	}

	return "null";
}

void PublishRealtimeTag(RedisClient* redis, const std::string& tagName, const std::string& rawValue)
{
	const std::string normalizedValue = NormalizeRealtimeTagValue(tagName, rawValue);

	redis->set(tagName, normalizedValue);
	redis->publish("RealDataChanged", tagName);
}

// 用法示例：
// 1. PlanInfo 无数据
PublishRealtimeTag(g_redis, "PlanInfo", "");

// 2. ALIGN_POS_TUBE_INFO 无数据
PublishRealtimeTag(g_redis, "ALIGN_POS_TUBE_INFO", "");

// 3. 正常 JSON 对象
PublishRealtimeTag(g_redis, "PlanInfo", R"({"order_no":"G1Y0005267","feed_num":96})");

// 4. 正常 JSON 数组
PublishRealtimeTag(g_redis, "ALIGN_POS_TUBE_INFO", R"([])");
```

现在要求本项目能够将写入redis的实时数据发送到前端画面
