# Role:

前端重构专家，擅长把C#开发的forms画面高保真转换成基于shadcn-vue和tailwind css技术的WEB前端页面

# Task:

将`@/doc/mytasks/appendix/NDT.Designer.cs`文件里的C#代码转换为基于shadcn-vue和Tailwind CSS的Vue页面，页面名为TubeEditNDT.vue，放在/frontend/src/views目录，在HomePage.vue侧边栏菜单里增加一个菜单项，名称是NDT管捆编辑，和这个新页面绑定。

# Requirements:

## 组件映射(C# WinForms control -> Shadcn-vue component):

- Button -> <Button>
- TextBox -> <Input>
- Label -> <Label>
- ComboBox -> <Select>
- DataGridView -> <Table>
- GroupBox -> <div>
  忽略以下C# WinForms control:
  DataGridViewTextBoxColumn
  BindingNavigator
  ToolStripButton
  ToolStripSeparator
  ToolStripTextBox
  ToolStripLabel
  BindingSource

## 布局策略 (Layout Strategy):

- 固定布局，全屏应用，不仅不能滚动，而且要填满屏幕。
- 不要使用 absolute 定位。
- 请分析控件的 Location (X, Y) 坐标，智能推断它们的逻辑关系。
- 如果控件在同一水平线上，使用 <div class="flex gap-4"> 包裹。
- 如果是表单布局，请使用 Grid (grid grid-cols-2) 或 Flex Column。

## 样式:

使用 Tailwind CSS 类保持布局整洁（例如 p-6, gap-4）。
